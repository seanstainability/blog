브라우저 저장소(Storage)
Web / JavaScript / Browser
2020.07.29.

브라우저 저장소(Storage)에 대해 설명해보세요.

쿠키

  • 만료 기한이 있는 키-값 저장소
  • document.cookie를 통해 현재 쿠키의 정보를 알아낼 수 있다.
  • 기본적으로 Stateless한 HTTP 프로토콜의 상태 유지를 위해 매 요청마다 서버로 쿠키가 같이 전송된다.
  • 4KB의 용량 제한이 있다.

로컬 스토리지, 세션 스토리지

  • 키-값 저장소
  • window.localStorage, window.sessionStorage
  • 둘 다 Storage 객체를 상속 받으며 length, key, getItem, setItem, removeItem, clear 메소드를 사용할 수 있다.
  • 서버에 자동으로 전송되지 않는다.
  • 로컬 스토리지 데이터는 사용자가 지우지 않는 이상 브라우저에 계속 남아있다.
  • 세션 스토리지 데이터는 브라우저 탭을 닫을 경우 제거된다.
